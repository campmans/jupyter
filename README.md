# MatCos Jupyter Notebooks

This repository contains Jupyter notebooks for the MatCos data science art project - a collaboration exploring the intersection of mathematics, data science, and creative visualization.

## About MatCos

MatCos is a collaborative project combining mathematical concepts with artistic visualization, creating dynamic art from data and algorithms.

## Getting Started

1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

2. Launch Jupyter:
   ```bash
   jupyter notebook
   ```

3. Navigate to the `notebooks/` directory to explore the notebooks.

## Project Structure

- `notebooks/` - Jupyter notebooks for data science art explorations
- `requirements.txt` - Python dependencies
